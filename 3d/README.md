TKK Game Collection

PC向けのブラウザゲーム集のテンプレートリポジトリ。

セットアップ

- ローカルで静的サーバーを立ててください。簡単な方法:
  - VS Code の Live Server で `3d` フォルダを開く
  - または Node.js があるなら `npx http-server .` を実行

ローカルでサーバーを使いたくない場合（file:// で完結させる方法）

- `index.html` をブラウザで直接開いてください（例: エクスプローラでダブルクリック）。
- ブラウザによっては iframe のセキュリティ制限があり、一部機能が制限される場合がありますが、このリポジトリでは親子ウィンドウは postMessage ベースで通信するように実装済みです。

構成

- `index.html` - ランチャー
- `styles.css` - 共通スタイル
- `src/` - JavaScript ソース
- `games/` - 各ゲームのフォルダ（テンプレート）

貢献と分担案

- 3人でそれぞれ2本ずつ実装担当すると効率的です（例: A=game1,2 B=game3,4 C=game5,6）。

ライセンス
